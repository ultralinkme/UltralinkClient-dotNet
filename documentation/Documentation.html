Class Documentation
MasterA kind of global class for the Master.
DatabaseThe core storage construct on a Master.
JobJobs assist humans in cleaning up databases.
UltralinkThe core data unit contained in a database.
WordWords are strings that ultralinks can potentially match against.
CategoryCategories are strings that are used for ultralink organization.
LinkLinks are hyperlinks and associated information that make up the bulk of an ultralink.
ConnectionA connection represents a relationship between two ultralinks.
PageFeedbackPage feedback values indicate an override for ultralinks on specific words on specific pages.
UserA class for users that have accounts registed on the Master.
AchievementAchievements for users to unlock.
